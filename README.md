# flexui
Huge update to FantasyUI, now named FlexUI
- Module settings to customize many aspects
    - Font/button size
    - Tab icons Large/small
- Compatibility for Foundry v9+
- Three different themes built in, choosable in module configuration
    - Fantasy
    - SciFi
    - Stone
